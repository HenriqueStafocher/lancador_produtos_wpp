# 🚀 Lançador de Produtos para WhatsApp

> Feito com ❤️ especialmente para minha mãe utilizar em seu grupo de brechó.

---

## 📌 Sobre o Projeto

O **Lançador de Produtos para WhatsApp** é uma aplicação web desenvolvida para facilitar a divulgação de produtos de brechó diretamente em grupos do WhatsApp. Com uma interface simples e intuitiva, basta adicionar o produto com fotos, título e descrição — e compartilhar no grupo com um clique!

---

## ✨ Funcionalidades

- 📸 **Upload de imagens** — suporte a 1, 2 ou 3 fotos por produto (JPEG ou PNG)
- 📝 **Cadastro de produtos** — título e descrição
- 💾 **Armazenamento persistente** — produtos salvos no banco de dados MySQL
- 🗂️ **Listagem de produtos disponíveis** — visualize todos os produtos cadastrados
- ✅ **Marcar como vendido** — remove o produto do banco ao ser vendido
- 📲 **Compartilhar no WhatsApp** — abre o WhatsApp Web com título, descrição e imagem pré-preenchidos
